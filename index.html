About

What if the Western Regional Technical Service Center ran their streamflow analysis in 1987 — and the code was still running today?

SIERRA-FLOW is a COBOL program that reads live USGS discharge data, computes per-station statistics, flags threshold alerts, calculates percent-of-normal vs historical medians, determines 30-day flow trends, groups stations into watershed basins, and produces a formatted 132-column report. This page reads streamflow.csv — the same file COBOL processed — and renders everything live. Data updates every morning at 7AM Pacific via GitHub Actions. COBOL doesn't apologize.
