Program Description

What if the Western Regional Technical Service Center ran their streamflow analysis in 1987 — and the code was still running today?

SIERRA-FLOW is a COBOL program that ingests USGS stream gage discharge data in CSV format, computes per-station statistics, flags threshold alerts for abnormal flow conditions, and produces a formatted columnar report — exactly as a water resources data center might have done four decades ago.

Built for fun. Built to run. Compiled in seconds. Processes 96 records across 8 Sierra Nevada gages in less time than it takes to boot a web framework. COBOL doesn't apologize.

Build & Run
# Install GnuCOBOL $ sudo apt install gnucobol # Ubuntu/WSL $ brew install gnucobol # macOS # Clone and run $ git clone https://github.com/bdgroves/sierra-flow-cobol $ cd sierra-flow-cobol $ cobc -x -o sierra-flow SIERRA-FLOW.cob $ ./sierra-flow $ cat streamflow-report.txt # Windows (after installing GnuCOBOL MinGW from arnoldtrembley.com) > cobc -x -o sierra-flow SIERRA-FLOW.cob > sierra-flow.exe # GitHub Actions runs this automatically on every push
Program Structure
IDENTIFICATION DIVISION Program metadata, author, date ENVIRONMENT DIVISION File assignments — INPUT csv / OUTPUT report DATA DIVISION FILE SECTION FD records for CSV and report files WORKING-STORAGE All variables, accumulators, 132-col report lines PROCEDURE DIVISION 0000-MAIN Orchestrates all paragraphs 1000-INITIALIZE Open files, capture run date 2000-PROCESS-FILE Main sequential read loop 2100-PROCESS-RECORD Route header vs. data records 2200-PARSE-CSV Character-by-character field splitter 2300-VALIDATE-RECORD Skip blanks and non-numeric values 2400-ACCUMULATE Per-station sum/min/max + alert count 3000-COMPUTE-STATS Mean = SUM / COUNT per station 4000-WRITE-REPORT Drive all three report sections 4100-WRITE-BANNER Title block with processing date 4200-WRITE-STATION-TABLE Section I: statistics 4300-WRITE-ALERT-SECTION Section II: threshold flags 4400-WRITE-SUMMARY Section III: run totals 4500-WRITE-FOOTER Closing banner 9000-TERMINATE Close files, print completion
